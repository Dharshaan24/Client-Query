# Client Query Management System

A Streamlit-based project for managing, tracking, and closing client queries.  
This system uses SQLite for data storage and Pandas for data management.

## Features
- Login & Registration (Client & Support)
- Query submission (Client)
- Query tracking and closure (Support)
- Query performance and status overview

## How to Run
```bash
pip install -r requirements.txt
streamlit run client_query_management_app.py
```
